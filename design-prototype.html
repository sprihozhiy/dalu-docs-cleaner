# Authentication

To interact with our API, you must include a Bearer token in the `Authorization` header of all requests.

## Generating an API Key

1. Navigate to the **Developer Settings** in your dashboard.
2. Click on Create New API Key.
3. Select the required scopes (Read/Write).
4. Copy the generated secret.

// Example Request
curl -X GET "https://api.example.com/user" \
  -H "Authorization: Bearer YOUR_API_KEY"
## Scopes

- `user.read`: Read profile information.
- `user.write`: Update profile information.
- `billing.read`: Access invoice history.
